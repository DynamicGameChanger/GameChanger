What is Game Changer?

Game Changer is a framework that allows an admin to dynamically control the settings of games while their client is actively playing.

Admins will be able to track progress of their clients through graphical data. Admins will be able to adjust difficulty according to the clients performance.

Game Changer takes variables in the form of an object to create buttons and knobs. As long as the game is created through JavaScript, you will be able to use Game Changer.
